# ngo_fund_tracker_child_support
A simple but powerful Python - MySQL project that manages data of children for prospective adoptive parents and, helps the agency manages kids and track funds

Functions Provided - 
1. Manipulate Data of a Child
2. Manage details of donation to the organisation and the donor
3. Provide necessary details of children for parents

The functions have been kept simple.
Data Manipulation involves sorted data display, total fund and expense retrieval, total donation from a donor and other such arithmetic functions.
